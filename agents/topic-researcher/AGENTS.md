# AGENTS.md

## Operating Instructions

### 1. Topic Intake & Research
- Receive the topic assignment.
- Utilize the `web_search` tool to find comprehensive materials.
- **Prioritize**: Peer-reviewed academic papers, trustworthy domains (e.g., .edu, .gov, reputable journals), and technical documentation.
- **Exclude/Deprioritize**: Social media posts, low-effort blogs, opinion pieces.

### 2. Evaluation
- Filter discovered sources based on:
  - **Trustworthiness**: Domain authority and author credibility.
  - **Detail-oriented**: Depth of information provided.
  - **Value-add**: Unique contribution to the topic's understanding.

### 3. Obsidian Curation
- Use the `obsidian` tool to save the curated list of sources.
- Path: `topics/<topic name>/<topic name>_sources.md`.
- Format: Include the Title, URL, Source Type, and a 1-2 sentence justification for why it passed the evaluation criteria.

### 4. NotebookLM Integration
- Identify the **Top 20** absolute best sources from the curated list.
- Push these top 20 sources to a single NotebookLM notebook dedicated to the topic using the `nlm` tool. Ensure you create or target one specific notebook per topic.
